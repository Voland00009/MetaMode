# MetaMode — Spec Фазы 1

**Дата:** 2026-04-06
**Статус:** РЕАЛИЗОВАНО (2026-04-06)

## Цель Фазы 1

Превратить CC из "исполнителя команд" в наставника. После установки CC:
- Объясняет *зачем*, а не только *как*
- Предлагает лучший путь, если видит
- Задаёт структурированные вопросы перед работой
- Не перегружает контекстное окно

## Что создаём (3 компонента)

### Компонент A: Менторский протокол (CLAUDE.md)

**Файл:** `MetaMode/dist/claude-md-addon.md`
**Куда устанавливается:** копируется в глобальный `~/.claude/CLAUDE.md` или в проектный `CLAUDE.md`

**Содержимое — инструкции для CC:**

1. **Режим наставника** — не выполняй молча, объясняй решения:
   - Перед действием: "Я сделаю X потому что Y" (1 строка, не эссе)
   - На развилке: "Есть вариант A и B. Рекомендую A потому что..."
   - При ошибке пользователя: мягко скажи, не просто делай как сказали

2. **Калибровка по уровню** — инструкции зависят от уровня пользователя:
   - Начинающий: объясняй термины, давай контекст, показывай альтернативы
   - Средний: объясняй только нетривиальное
   - Продвинутый: минимум объяснений, максимум действий
   - Уровень определяется один раз и сохраняется в memory

3. **Когда НЕ менторить** — чтобы не раздражать:
   - Простые команды (коммит, форматирование, переименование) — просто делай
   - Повторные задачи — не объясняй то же самое дважды
   - Пользователь явно сказал "просто сделай" — подчинись

**Размер:** ~30-40 строк. Больше нельзя — съест контекст.

---

### Компонент B: Скилл "Умные вопросы" (Superpowers skill)

**Решение:** объединён с компонентом A в один файл `claude-md-addon.md`.

Причина: отдельный Superpowers skill потребовал бы npm-пакет (лишняя инфраструктура и расходы). Инструкции в CLAUDE.md дают тот же результат бесплатно. Скилл можно выделить позже, когда понадобится.

**Как работает (встроен в менторский протокол):**
- CC сам предлагает вопросы перед сложной задачей
- Показывает оценку сложности и рекомендуемое количество
- Вопросы с вариантами ответов (a/b/c/d/свой) — из Prompt Booster
- Пользователь может сказать "задай вопросы" в любой момент

### Компонент C: Архитектура файлов (каркас)

**Структура MetaMode после Фазы 1:**

```
MetaMode/
├── docs/                          # документация проекта
│   ├── vision_v2_full.md          # (существует)
│   ├── critical_analysis.md       # (существует)
│   └── phase1_spec.md             # (этот файл)
├── input/                         # (существует) входные материалы
├── dist/                          # ПРОДУКТ — то что устанавливается
│   ├── claude-md-addon.md         # менторский протокол + вопросы (всё в одном)
│   └── install.md                 # инструкция по установке
└── CLAUDE.md                      # инструкции для разработки MetaMode
```

**Принцип "цепочка файлов":**
- `CLAUDE.md` пользователя содержит только 2-3 строки про MetaMode
- Эти строки указывают на менторский протокол
- Протокол работает сам, скилл вызывается по имени
- Ничего лишнего не загружается в контекст

## Формат доставки (как пользователь установит)

**Фаза 1 — ручная установка (3 шага):**

1. Добавить 2 строки-указателя в свой `CLAUDE.md`
2. Создать файл уровня в memory (или CC спросит сам)
3. Готово — протокол и вопросы работают через CLAUDE.md

**Почему не автоматизируем:** это Фаза 1, один пользователь. Автоматизация — лишняя сложность сейчас.

## Решённые вопросы (2026-04-06)

1. **Локальный скилл vs npm-плагин?**
   → Ни то, ни другое. Всё через CLAUDE.md инструкции. Причина: npm-пакет — лишняя инфраструктура и расходы. CLAUDE.md работает бесплатно и сразу. Скилл можно выделить позже.

2. **Калибровка уровня — где хранить?**
   → Memory файл (`user_level.md`). Уже загружается автоматически.

3. **Триггер вопросов — автоматический или ручной?**
   → Оба. CC сам предлагает через инструкцию в протоколе. Пользователь может попросить в любой момент.
