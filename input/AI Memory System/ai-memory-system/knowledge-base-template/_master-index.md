# Knowledge Base — Master Index

> Глобальная база знаний. Уроки, паттерны и ошибки из всех проектов.
> Агент начинает навигацию отсюда.

---

## Правила для агента

### Навигация
1. Прочитай этот файл → список категорий
2. Перейди в нужную категорию через wiki-ссылку
3. Прочитай `_index.md` категории → список статей
4. Читай конкретные статьи по необходимости

### Запись нового урока (через /reflect)
1. Найди подходящую категорию
2. Проверь дубли в `_index.md`
3. Запиши урок, обнови индексы

### Правила категорий
- Категория = **технология** (supabase, nextjs, n8n) или **домен** (design, deployment, content)
- НЕ по типу ошибки (не "common-mistakes")
- НЕ по проекту (не "my-app-lessons")

### Правила файлов
- Имя: kebab-case, описательное
- Обязательно: frontmatter с tags, source-project, date
- Обязательно: wiki-ссылки на связанные файлы
